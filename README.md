# Python Calculator

Простой калькулятор на Python — проект для лабораторной работы по Git.

## Файлы

- `calculator.py` — основные арифметические операции
- `utils.py` — вспомогательные математические функции
- `main.py` — точка входа в программу

## Запуск

```bash
python main.py
```

## Команды Git использованные в проекте

```bash
git init
git add .
git commit -m "Initial commit"
git branch feature
git checkout feature
git merge feature
git push origin main
```
